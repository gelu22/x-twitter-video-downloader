Privacy Policy

Extension: X.com Video Downloader
Last updated: February 2026

1. What this extension does

X.com Video Downloader is a browser extension that allows users to download publicly available videos from X.com (formerly Twitter) directly to their device.

2. Data we collect

We collect no personal data. The extension does not transmit any information to the developer or any third-party server owned by the developer.

3. Clipboard access (clipboardRead)

When the extension popup is opened, it reads the clipboard once solely to detect whether a tweet URL is already copied. If a valid X.com/twitter.com link is found, it is automatically pasted into the input field for convenience. This data is processed entirely locally in the browser and is never sent anywhere.

4. Network requests

The extension makes a single request to the public Twitter/X Syndication API (cdn.syndication.twimg.com) to retrieve publicly available video metadata for the tweet URL provided by the user. No user identifiers or cookies are attached to this request beyond what the browser sends automatically.

5. Downloads

Files are downloaded directly from X's CDN (*.twimg.com) to the user's device using the browser's built-in download mechanism. The developer has no access to downloaded files.

6. Third-party services

The extension itself does not integrate any analytics, advertising, or tracking services.

7. Children's privacy

This extension is not directed at children under 13. We do not knowingly collect data from children.

8. Changes to this policy

Any updates will be reflected on this page with a new "Last updated" date.

9. Contact

If you have questions about this privacy policy, please open an issue on the GitHub repository or contact the developer via the Chrome Web Store listing.
